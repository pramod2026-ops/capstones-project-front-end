# Quiz App

A simple quiz application built using HTML, CSS, and JavaScript.

This project displays quiz questions with multiple answer choices. Users can answer questions, see their score, and restart the quiz after finishing.

## Features

* Start quiz screen
* Multiple choice questions
* Score tracking system
* Progress bar
* Result screen
* Restart quiz button
* Answer validation using JavaScript
* Dynamic question rendering

## Technologies Used

* HTML
* CSS
* JavaScript
